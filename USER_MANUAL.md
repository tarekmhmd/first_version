## Getting Started

### 1. First Time Setup

1. **Run Setup**
   - Double-click `setup.bat`
   - Wait for installation to complete (5-15 minutes)
   - You'll see "Setup completed successfully!"

2. **Start Application**
   - Double-click `run.bat`
   - Your browser will open automatically
   - If not, go to: http://localhost:5000

### 2. Create Your Account

1. Click the **Register** tab
2. Fill in your information:
   - Full Name
   - Email Address
   - Password
   - Age (optional)
   - Address (optional)
3. Click **Register**
4. You'll see "Registration successful!"

### 3. Login

1. Enter your email and password
2. Click **Login**
3. You'll be redirected to your Dashboard

## Understanding Results

### Severity Levels

- **None**: Healthy, no issues detected
- **Mild**: Minor condition, self-care possible
- **Moderate**: May need medical attention
- **Severe**: Seek medical attention immediately

### Confidence Levels

- **80-100%**: High confidence
- **60-79%**: Moderate confidence
- **Below 60%**: Low confidence, verify with doctor

## Troubleshooting

### Application won't start
- Make sure you ran `setup.bat` first
- Check if Python 3.11.4 is installed
- Try running as Administrator

### Can't login
- Check email and password
- Make sure you registered first
- Try resetting (contact support)

### Analysis fails
- Check image/audio quality
- Ensure file format is supported
- Try a different file
- Check internet connection

### Slow performance
- Close other applications
- Restart the application
- Check available disk space
- Update to latest version
